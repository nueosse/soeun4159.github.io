커리어패스 탐색 미션

진로에 중요한 가치와 우선순위를 정하고 시장 분석을 통해 인사이트를 가지고, 희망 직무와 관련된 채용 공고를 분석하며 현재 시장에서 요구하는 역량을 파악하기

개요

이번 커리어패스 탐색 미션을 통해 내가 중요하게 생각하는 진로 가치와 우선순위를 정리하고, 희망 직무와 관련된 채용 공고를 분석하며 현재 시장에서 요구하는 역량을 파악하고자 한다. 이를 바탕으로 앞으로 어떤 방향으로 준비해야 할지 구체적인 계획을 세우는 것을 목표로 한다.
시장 분석 → 희망 회사(3개) → 희망 직무 → 내가 지금 할 것

진로 가치 선택

연봉, 근무지역, 워라밸, 안정성, 성장가능성, 직무적합성, 자율성, 함께 일하는 사람, 조직문화, 명예, 회사 인지도, 재미

가장 중요한 가치 2개

  • 워라밸
    지속적인 성장과 자기계발을 위해 충분한 개인 시간을 확보하는 것이 중요하기 때문이다.
  • 연봉
    노력에 대한 정당한 보상과 안정적인 환경이 커리어 집중도를 높일 수 있다고 생각했다.

우선순위가 낮은 가치 2개

  • 명예
    외부 평가보다 실제 성장과 실력 향상이 더 중요하다고 판단했기 때문이다.
  • 회사 인지도
    기업의 유명세보다 직무 경험과 성장 가능성이 더 중요하다고 생각한다.

시장 분석

희망 회사 3개

  1. 경찰청 사이버수사국 : 포렌식 기반 수사 업무
  2. SK쉴더스 : 기업 보안 실무 경험
  3. 토스뱅크 : 웹 및 클라우드 기반 보안

공고 5개

공고 1

회사 / 분야 : 티오리 : Offensive Security Researcher

👉 공격자 관점에서 시스템에 침투 가능한 경로를 분석하는 오펜시브 보안 경험과, PoC 및 Exploit 작성 능력, 그리고 리버싱을 통한 취약점 분석 역량이 중요하게 요구되었다.

링크: https://careers.theori.io/ko/o/16295

공고 2

회사 / 분야 : 엔키화이트햇 RedOps팀 : 화이트해커 인턴

👉 이 공고는 웹 해킹, 리버싱, 포너블 등 다양한 취약점 분석 경험과 Python 기반 PoC 개발 능력, 그리고 MITRE ATT&CK 프레임워크에 대한 이해가 중요하게 요구되었다.

링크: https://enki.career.greetinghr.com/ko/o/199221

공고 3

회사 / 분야 : 토스뱅크 Security Researcher

👉 이 공고는 웹 및 API 취약점 분석 경험과 스크립트 기반 자동화 능력, 그리고 클라우드 환경과 MSA 아키텍처에 대한 이해를 바탕으로 한 실무형 보안 역량이 중요하게 보인다.

링크: https://toss.im/career/job-detail?job_id=4076085003&sub_position_id=5847765003&company=토스뱅크

공고 4

회사 / 분야 : sk쉴더스 보안관제

👉 이 공고는 방화벽, IDS/IPS 등 보안 장비에 대한 이해와 로그 분석을 기반으로 한 침해사고 대응 경험, 그리고 네트워크 및 시스템 보안에 대한 실무 경험이 중요하게 보였다.

링크: https://www.skshieldusapply.com/ko/o/107003

공고 5

회사 / 분야 : 경찰청 경채 사이버수사

👉 이 공고는 디지털 포렌식 역량을 요구하며, 로그 및 파일 시스템 분석 경험이 중요하게 보였다.

링크: https://public.jinhakapply.com/policev2/public/public_3.aspx

반복해서 나온 역량 3가지

  • 웹 및 시스템 취약점 분석 능력

    다양한 공고에서 웹 취약점, API 취약점, 소스코드 오디팅, 리버싱 등 여러 형태의 취약점 분석 역량이 공통적으로 요구되었다. 이는 단순한 도구 사용을 넘어, 공격자 관점에서 취약점의 원인을 이해하고 재현할 수 있는 능력이 중요하다는 것을 의미한다.

  • Python 기반 PoC 개발 및 자동화 역량

    여러 공고에서 Python, Shell 등을 활용한 스크립트 작성 능력과 PoC 개발 경험이 요구되었다. 이는 취약점을 발견하는 것에서 끝나는 것이 아니라, 코드로 검증하고 자동화할 수 있는 개발 역량이 필수적이라는 것을 보여준다.

  • 시스템 및 로그 기반 분석 능력

    보안 관제와 포렌식 관련 공고에서 로그 분석과 시스템 및 네트워크 구조에 대한 이해가 공통적으로 강조되었다. 이는 공격 기술뿐만 아니라 시스템 내부 동작을 이해하고 이상 징후를 분석할 수 있는 능력이 핵심 역량임을 의미한다.

내가 지금 해야할 것

웹 취약점 분석을 중심으로 학습을 진행하고, Python 기반 PoC 작성 및 자동화 역량을 강화하는 것이 필요하다. 또한 로그 분석과 시스템 및 네트워크 구조 이해를 병행하여, 단순한 공격 기술 습득을 넘어 실제 환경에서 분석과 대응이 가능한 실무형 보안 역량을 갖추는 것을 목표로 해야겠다.

마무리

이번 탐색 과정을 통해 내가 중요하게 여기는 가치와 실제 채용 시장에서 요구하는 역량을 비교해볼 수 있었고, 앞으로 어떤 방향으로 준비해야 할지 더 구체적으로 생각해보는 계기가 되었다.
